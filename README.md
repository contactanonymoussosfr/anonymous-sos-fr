# 🔴 Anonymous SOS FR

## Plateforme communautaire sécurisée et surveillée

Bienvenue sur *Anonymous SOS FR*, une plateforme communautaire française dédiée aux échanges, aux lives et à la radio, dans un cadre strictement encadré, sécurisé et surveillé.

---

## 🎯 Objectif du projet

Anonymous SOS FR a pour but de proposer :
- une plateforme gratuite et accessible,
- des lives publics et privés encadrés,
- une radio facultative,
- une communauté respectueuse,
- une surveillance permanente.

L’utilisation du site implique l’acceptation totale des règles.

---

## 🔊 Radio

- La radio est facultative  
- Chaque utilisateur peut l’activer ou la couper à tout moment  
- Aucun son n’est imposé  

---

## 👤 Comptes

- La création d’un compte est obligatoire pour accéder aux fonctionnalités  
- L’adresse e-mail doit être vérifiée  
- Plusieurs comptes sont autorisés  
- Le site est surveillé en continu  

---

## 🚫 VPN

- Les VPN sont interdits pour les utilisateurs  
- Les VPN sont autorisés uniquement pour les personnes travaillant sur le site  
- VPN détecté = accès refusé  

---

## 🔴 Lives

- Une demande est obligatoire pour faire un live  
- Le type de live doit être précisé  
- Une vérification d’identité est exigée  
- Sans validation, aucun live ne peut être lancé  

---

## 🔐 Lives privés

- Les lives privés sont sur invitation uniquement  
- Un abonnement est obligatoire  
- Le service client doit être contacté avant le live  
- L’équipe du site peut rejoindre tous les lives privés  

---

## 🛡️ Modération

- Tous les lives sont surveillés  
- Les modérateurs peuvent muter, exclure ou bannir  
- Toute sanction est justifiée et limitée dans le temps  

---

## 🎁 Cadeaux & XP

- Les cadeaux rapportent de l’XP  
- 1 € dépensé = 1 XP  
- Les niveaux permettent d’obtenir des badges  
- Niveau 50 = badge spécial
